# Print-It WebSite v.1 : Dynamisation du site (Projet P6 OpenClassrooms)
Ce projet consiste à dynamiser le site Internet statique d’une petite imprimerie familiale, **Print it**, afin d’améliorer l’expérience utilisateur grâce à un carrousel entièrement fonctionnel en JavaScript.

Il s’agit du **Projet 6** du parcours Développeur Web chez **OpenClassrooms**.

# Objectifs du projet
- Manipuler le DOM en JavaScript  
- Ajouter des **Event Listeners**  
- Gérer un **slider/carrousel fonctionnel**  
- Implémenter les interactions utilisateur  
- Rendre le site plus dynamique et agréable à utiliser  

# Fonctionnalités implémentées

Étape 1 : Mise à jour du code HTML
- Ajout des éléments nécessaires pour accueillir le slider  
- Organisation du DOM pour permettre le dynamisme  

Etape 2 : Ajout des Event Listeners sur les flèches
- Navigation entre les slides avec les boutons précédent et suivant  

Etape 3 : Ajout des bullet points au slider
- Génération automatique des bullets en fonction du nombre d’images  
- Mise à jour visuelle de la bullet active  

Étape 4 : Changement de slide via les bullets
- Possibilité pour l’utilisateur de cliquer sur une bullet pour changer de slide  

Étape 5 : Défilement infini (carrousel en boucle)
- Lorsque l’utilisateur atteint la dernière image, le slider revient automatiquement au début  
- Fonctionne également dans le sens inverse
